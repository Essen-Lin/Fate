# Fate Datebase Generation


## 使用步驟
### 安裝chromdriver
1. 確認現在使用chrome版本

     - 在電腦上開啟Chrome。
     - 按一下右上方的 **「更多」** 圖示 。
     - 依序按一下 **[說明]** **[關於Google Chrome]**。

2. 根據 **步驟1** 所顯示的chrome版本，下載對應的chromedrive放到資料夾中
    - 透過 https://chromedriver.chromium.org/downloads 下載對應版本的chromedrive
    - 放到執行程式的資料夾中 (複製路徑)
    - 將411 行 PATH 改成剛剛複製的路徑

### 安裝Python 3.9
